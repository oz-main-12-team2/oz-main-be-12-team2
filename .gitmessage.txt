 ```markdown
## Commit Title Type

- ✨ **feat** : 새로운 기능 추가
- 🐛 **fix** : 기능 수정, 버그 수정
- 💡 **chore** : 오타 수정 및 새로운 기능이 추가되지 않고, 코드가 변경된 경우 (주석 추가 및 수정 포함)
- 📝 **docs** : 문서 수정 (예: README 수정)
- 🚚 **build** : 빌드 관련 파일 수정 및 삭제한 경우
- ✅ **test** : 테스트 코드, 리팩터링 테스트 코드 추가 (프로덕션 코드 변경 없음)
- ♻️ **refactor** : 코드 리팩터링
- 🚑 **hotfix** : 긴급 수정
```
